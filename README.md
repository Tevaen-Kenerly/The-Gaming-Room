# The-Gaming-Room
A web-based multiplayer game application designed with a Software Design Document, UML class modeling and object-oriented principles. This is the first prototype showcasing how The Gaming Room can offer their game, Draw It or Lose It, in a web-based format and serve multiple platforms.
# Overview
The Gaming Room is a web-based multiplayer game application designed with a strong emphasis on software design principles and object-oriented programming (OOP). The project demonstrates how structured UML modeling, design constraints, and efficient system architecture can be applied to create scalable, maintainable, and user-friendly applications.
